1) install first express-generator using npm globally (node should be installed and npm)
    $ npm install express-generator -g

2) create the project (for this example 'TEST')
    $ express TEST

3) after the project has been created go the folder
    $ cd TEST

4) install the needed dependencies using npm install
    $ npm install

5) to start the project, go to URL http://localhost:3000
    $ npm start
